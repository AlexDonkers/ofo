Occupant Property Taxonomy

The occupant property taxonomy (OPT:) is a small taxonomical extension of OFO. The extension is openly available at https://github.com/AlexDonkers/opt. The OPT ontology is a taxonomy of properties related to occupants and their comfort and may be extended based on new use-cases. The authors welcome anyone to suggest extensions.

SPARQL

Querying feedback given by John Doe

PREFIX bop: <https://w3id.org/ofo#>
PREFIX bot: <https://alexdonkers.github.io/opt#>

SELECT ?comfortProperty ?feedback ?datapoint ?database

WHERE {
:JohnDoe ofo:givesFeedback ?feedback .
?feedback ofo:hasEvaluatedProperty ?comfortProperty .
?comfortProperty a ofo:Property .

?feedback ofo:hasResult ?datapoint .
?datapoint ofo:isDataPointOf ?database .
}

USE ?database
SELECT ?feedback FROM ?datapoint ORDER BY time LIMIT 1

DOWNLOADS

Open Flat

The Open Flat is an initiative by Alex Donkers to create more open-source examples of linked building data.
The model represents an apartment in The Netherlands.


Open Smart Home

The Open Smart Home is an initiative by Schneider and Rasmussen. The building represents an apartment in Nuremberg, Germany. We converted the revit file to DAE and GLTF formats, and manually created a TTL file of the building that follows the BOP-ontology.
Various sensors measured the temperature, relative humidity, brightness and outdoor temperature for a few months. This data is available in CSV format.
